﻿# Users Table

Простое React-приложение для отображения пользователей из API `https://dummyjson.com/users`.

## Запуск

```bash
npm install
npm start
```

## Сборка

```bash
npm run build
```

## GitHub Pages

1. Укажите правильный `homepage` в `package.json`.
2. Выполните:

```bash
npm run deploy
```
